# geom4py v0.1 規劃

## 概述

`geom4py` 是一個 Python 幾何學基礎套件，提供向量、點、直線、平面等基本幾何物件的運算功能。

## 目標

- 輕量級、依賴最少（僅需 NumPy）
- API 直覺易懂
- 適合教學與科研使用
- 發布至 PyPI

```python
import geom4py as geom

```

## 功能規劃

### Vector (向量)

| 方法 | 說明 |
|------|------|
| `Vector(x, y, z)` | 建立三維向量 |
| `+`, `-` | 向量加減 |
| `*`, `/` | 向量與純量乘除 |
| `dot(v)` | 點積 |
| `cross(v)` | 叉積 |
| `norm()` | 向量長度 |
| `normalize()` | 單位向量 |
| `angle_to(v)` | 與另一向量夾角 |

### Point (點)

| 方法 | 說明 |
|------|------|
| `Point(x, y, z)` | 建立三維點 |
| `+`, `-` | 點+向量、點-點=向量 |
| `distance_to(p)` | 與另一點距離 |
| `to_vector()` | 轉換為原點向量 |

### Line (直線)

| 方法 | 說明 |
|------|------|
| `Line(point, direction)` | 過點沿方向建立直線 |
| `Line.from_points(p1, p2)` | 過兩點建立直線 |
| `point` | 直線上任意點 |
| `direction` | 方向向量 |
| `distance_to_point(p)` | 點到直線距離 |
| `closest_point(p)` | 直線上離點最近的點 |

### Plane (平面)

| 方法 | 說明 |
|------|------|
| `Plane(point, normal)` | 過點以法向量建立平面 |
| `Plane.from_points(p1, p2, p3)` | 過三點建立平面 |
| `normal` | 法向量 |
| `point` | 平面上任意點 |
| `distance_to_point(p)` | 點到平面距離 |
| `project_point(p)` | 點投影到平面 |

## 數學公式

### 點到直線距離
```
d = ||(P - P0) × d|| / ||d||
```

### 點到平面距離
```
d = |(P - P0) · n| / ||n||
```

## API 範例

```python
import geom4py as geom

# 向量運算
v1 = geom.Vector(1, 0, 0)
v2 = geom.Vector(0, 1, 0)
print(v1.cross(v2))  # Vector(0, 0, 1)

# 點到直線距離
line = geom.Line.from_points(geom.Point(0,0,0), geom.Point(1,0,0))
p = geom.Point(0, 1, 0)
print(line.distance_to_point(p))  # 1.0

# 點到平面距離
plane = geom.Plane(geom.Point(0,0,0), geom.Vector(0,0,1))
p = geom.Point(1, 1, 1)
print(plane.distance_to_point(p))  # 1.0
```
