# 安全審查者測試（含目錄存取測試）

## 原理

此測試工具在 `_more/v3-agent-reviewer/test_reviewer.md` 的基礎上，增加了**目錄存取控制測試**。它不僅測試 LLM 安全審查模型，還測試 `check_outside_access()` 函數是否能正確偵測外部路徑存取。

### 測試類型

1. **安全命令測試**：驗證無害命令被標記為 SAFE
2. **危險命令測試**：驗證破壞性命令被標記為 UNSAFE
3. **外部存取測試**：驗證目錄邊界檢查是否正確

### 外部存取測試用例

| 命令 | 所在目錄 | 預期結果 |
|------|---------|---------|
| `ls /tmp` | cwd | 需詢問 |
| `cat /etc/passwd` | cwd | 需詢問 |
| `cd .. && ls` | cwd | 需詢問 |
| `ls -la` | cwd | 不需詢問 |
| `cat file.txt` | cwd | 不需詢問 |
| `grep 'test' *.py` | cwd | 不需詢問 |

## 使用場景

- 整合測試安全審查 + 目錄控制
- 驗證雙層安全機制
- 調試正則表達式路徑檢測

## 優缺點

| 優點 | 缺點 |
|------|------|
| 整合雙層測試 | 正則檢測有限 |
| 自動化測試用例 | 無邊界案例測試 |
