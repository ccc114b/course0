# Chat0 — 純對話 LLM 客戶端（無工具）

## 原理

這是最基礎的 LLM 對話客戶端，僅實現 **純文字聊天** 功能，沒有任何工具使用或 Agent 能力。它使用 Ollama 的 `/api/chat` 端點，維護對話歷史列表，支援多輪對話。

### 與進階版本的對比

這是整個 Agent 系列中最簡單的版本：
- **無工具**：不能執行命令、讀寫檔案
- **無記憶**：僅在上下文維護對話歷史
- **無安全審查**：不控制 LLM 的輸出
- **單一模型**：只有一個 LLM 聊天

### API 差異

使用 Ollama 的 `chat` API（傳入 messages 列表）而非 `generate` API（傳入單一 prompt），這是更標準的 LLM API 格式。

## 使用場景

- 教學演示：從純聊天到完整 Agent 的演進起點
- 測試 Ollama 連線是否正常
- 簡單的問答需求

## 優缺點

| 優點 | 缺點 |
|------|------|
| 最簡潔的實作 | 無任何工具能力 |
| 易於理解 | 無記憶管理 |
| 標準 chat API | 無安全機制 |
