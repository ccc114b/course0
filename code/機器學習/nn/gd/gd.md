# 梯度下降法 (Gradient Descent)

## 原理

梯度下降法是一種一階疊代最佳化演算法，用於尋找可微分函數的局部最小值。核心概念是沿著函數梯度（斜率）的反方向逐步調整參數，因為梯度方向是函數值上升最快的方向，反方向即為下降最快的方向。

### 演算法步驟

1. 從初始點 $p_0$ 開始
2. 計算當前點的梯度 $\nabla f(p)$
3. 沿梯度反方向移動一小步：$p_{new} = p - \eta \nabla f(p)$
4. 重複直到梯度長度小於某閾值或達到最大疊代次數

### 數學公式

**梯度定義**:
$$\nabla f(p) = \left(\frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \dots, \frac{\partial f}{\partial x_n}\right)$$

**參數更新規則**:
$$p_{t+1} = p_t - \eta \nabla f(p_t)$$

其中 $\eta$ 為學習率（步伐大小）。

**數值梯度近似**（用於無法解析計算梯度的情況）:
$$\frac{\partial f}{\partial x_k} \approx \frac{f(p + h e_k) - f(p)}{h}$$

### 使用場景

- 機器學習中最小化損失函數
- 線性回歸參數估計
- 神經網路權重訓練
- 任何可微分函數的最佳化問題

### 優缺點

| 優點 | 缺點 |
|------|------|
| 實作簡單，容易理解 | 可能收斂到局部最小值而非全域最小值 |
| 計算成本低（每次僅需一次函數求值） | 在病態曲率區域收斂緩慢 |
| 適用於大規模問題 | 學習率需要手動調整 |
| 理論成熟，收斂性有保證 | 對特徵尺度敏感（需要標準化） |
