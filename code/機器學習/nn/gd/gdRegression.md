# 線性回歸之梯度下降 (Linear Regression with Gradient Descent)

## 原理

使用梯度下降法擬合一條直線 $y = a_0 + a_1 x$，使其盡可能符合給定的資料點 $(x_i, y_i)$。以均方誤差（MSE）作為損失函數，透過梯度下降法尋找最佳參數 $a_0$, $a_1$。

### 演算法步驟

1. 定義預測函數 $predict(a, x) = a_0 + a_1 x$
2. 定義均方誤差損失函數 $MSE(a) = \frac{1}{N}\sum_{i=1}^N (y_i - \hat{y}_i)^2$
3. 從初始點 $a = [0, 0]$ 開始
4. 對損失函數執行梯度下降
5. 繪製擬合結果與原始資料

### 數學公式

**線性模型**:
$$\hat{y}_i = a_0 + a_1 x_i$$

**均方誤差 (MSE)**:
$$MSE(a_0, a_1) = \frac{1}{N}\sum_{i=1}^N (y_i - (a_0 + a_1 x_i))^2$$

**參數梯度**:
$$\frac{\partial MSE}{\partial a_0} = -\frac{2}{N}\sum_{i=1}^N (y_i - \hat{y}_i)$$
$$\frac{\partial MSE}{\partial a_1} = -\frac{2}{N}\sum_{i=1}^N (y_i - \hat{y}_i) x_i$$

### 使用場景

- 連續數值預測（房價、溫度、銷售量）
- 趨勢分析與趨勢線擬合
- 機器學習入門教學
- 變數間線性關係的探索

### 優缺點

| 優點 | 缺點 |
|------|------|
| 模型簡單直觀，可解釋性強 | 只能捕捉線性關係 |
| 訓練速度快，計算成本低 | 對異常值敏感 |
| 理論完備，有封閉解（正規方程） | 需要特徵工程來處理非線性關係 |
| 梯度下降版本可擴展到大型資料集 | 若學習率不當可能不收斂 |
