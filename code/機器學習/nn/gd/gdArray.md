# 多變量梯度下降範例 (Gradient Descent with Array)

## 原理

本範例展示如何使用梯度下降法尋找三元二次函數 $f(x,y,z) = (x-1)^2 + (y-2)^2 + (z-3)^2$ 的最小值。該函數為凸函數，全域最小值位於點 $(1,2,3)$，梯度下降法可以保證收斂到該點。

### 演算法步驟

1. 定義目標函數 $f(p)$，其中 $p = [x, y, z]$
2. 設定初始點 $p_0 = [0, 0, 0]$
3. 呼叫通用梯度下降函數 `gradientDescendent`
4. 輸出最終找到的最小值點

### 數學公式

**目標函數**:
$$f(x,y,z) = (x-1)^2 + (y-2)^2 + (z-3)^2$$

**梯度**:
$$\nabla f = (2(x-1),\; 2(y-2),\; 2(z-3))$$

**最小值**:
$$\arg\min f(x,y,z) = (1, 2, 3)$$

### 使用場景

- 示範多變量最佳化的基本概念
- 驗證梯度下降法在凸函數上的收斂性
- 教學用簡潔範例

### 優缺點

| 優點 | 缺點 |
|------|------|
| 凸函數保證收斂到全域最小值 | 僅適用於簡單的二次函數 |
| 可直接觀察到梯度下降的收斂過程 | 真實問題的函數通常更複雜 |
| 參數代碼簡潔易懂 | 收斂速度受初始點與學習率影響 |
