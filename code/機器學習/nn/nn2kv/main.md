# main.py — GPT 模型訓練與推論主程式（nn2kv KV Cache 版）

## 原理

本程式是 nn2kv 系列的主入口，使用 chargpt.py 的高階 API 進行訓練與推論。示範了具備 KV Cache 加速的完整語言模型 pipeline。

### 演算法步驟

1. 設定亂數種子（確保可重現）
2. 準備資料集與 Tokenizer
3. 初始化 GPT 模型與 Adam 優化器
4. 呼叫 `train_model` 進行訓練（含梯度裁剪）
5. 呼叫 `generate_samples` 進行推論（含 KV Cache）

### 使用場景

- 一鍵式語言模型訓練與推論
- 比較 nn2kv 與 nn1tensor 的推論速度差異
- 生產環境的文字生成應用

### 優缺點

| 優點 | 缺點 |
|------|------|
| 模組化設計，程式碼簡潔 | 隱藏了實作細節，不易於教學 |
| 訓練含梯度裁剪，更穩定 | 超參數固定，需修改原始碼調整 |
| 推論使用 KV Cache 加速 | 無中斷恢復機制 |
