# CNN — 卷積神經網路 (Convolutional Neural Network)

## 原理

此模組實作了一個簡易的卷積神經網路（CNN），包含卷積層（Conv2d）、最大池化層（MaxPool2d）、全連接層（Linear）以及完整的 CNN 模型。所有運算皆建立在 nn0.py 的自動微分引擎之上，支援反向傳播訓練。

### 核心架構

1. **Conv2d (卷積層)**: 使用滑動視窗對輸入影像進行卷積運算，提取區域特徵
2. **MaxPool2d (最大池化層)**: 對特徵圖進行降採樣，減少維度並保留最強烈的響應
3. **Linear (全連接層)**: 將展平後的特徵映射到類別空間
4. **CNN 模型**: 結合卷積、池化、全連接的完整分類網路

### 演算法步驟

1. 輸入影像經過 Conv2d 卷積 → ReLU 激活
2. 最大池化降採樣
3. 展平 (Flatten) 特徵圖
4. 全連接層輸出分類 logits

### 數學公式

**二維卷積**:
$$(I * K)(i,j) = \sum_{c} \sum_{u} \sum_{v} I_c(i+u, j+v) \cdot K_{oc, c}(u, v) + b_{oc}$$

**最大池化**:
$$MaxPool(I)(i,j) = \max_{u,v \in [0, k)} I(i \cdot s + u, j \cdot s + v)$$

**ReLU 激活函數**:
$$ReLU(x) = \max(0, x)$$

### 使用場景

- 影像分類（MNIST 手寫數字辨識）
- 區域特徵提取
- 卷積神經網路教學範例

### 優缺點

| 優點 | 缺點 |
|------|------|
| 卷積層可有效捕捉空間區域特徵 | 純 Python 標量引擎訓練極慢 |
| 池化層降低維度，減少參數量 | 無批次處理（batch）支援 |
| 完整的自動微分支援 | 卷積實作為雙重迴圈，效率低 |
