# 線性分類器與 Adam 優化器範例

## 原理

使用線性分類器（單層 softmax 回歸）進行二元分類。模型使用交叉熵損失函數，搭配 Adam 優化器進行參數更新。範例展示完整的 forward → loss → backward → update 訓練循環。

### 演算法步驟

1. 初始化權重矩陣 W (2×3) 與 Adam 優化器
2. 對每個訓練步：
   - 前向傳播計算 logits: $logits = W \cdot x$
   - 計算交叉熵損失
   - 反向傳播更新權重
3. 觀察 logits 變化，確認模型學會正確分類

### 數學公式

**線性分類器**:
$$logits = W \cdot x + b$$

**交叉熵損失**:
$$Loss = -\log\left(\frac{e^{logits_{target}}}{\sum_j e^{logits_j}}\right)$$

### 使用場景

- 二元分類問題
- 展示 Adam 優化器的收斂過程
- 教學示範：從隨機權重到正確分類
