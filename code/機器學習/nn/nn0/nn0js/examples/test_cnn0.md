# 使用 nn0.js 訓練 MNIST 手寫數字辨識

## 原理

使用 nn0.js 自動微分引擎實作一個多層感知器（MLP）來辨識 MNIST 手寫數字。網路結構為：輸入層 784 → 隱藏層 16 (ReLU) → 輸出層 10。由於純 JS 標量運算的速度限制，僅使用少量資料進行示範。

### 核心架構

1. **LinearLayer**: 全連接層，包含權重與偏差
2. **MNISTModel**: 兩層 MLP 網路
3. **MNIST 資料載入**: 從二進位檔案讀取影像與標籤

### 演算法步驟

1. 載入 MNIST 資料（或使用隨機假資料）
2. 前向傳播計算 logits
3. 交叉熵損失計算
4. 反向傳播與 Adam 更新
5. 計算準確率

### 數學公式

**MLP 網路**:
$$h = ReLU(W_1 \cdot x + b_1)$$
$$logits = W_2 \cdot h + b_2$$

### 使用場景

- 手寫數字辨識教學範例
- 展示自動微分在實際 ML 問題中的應用
- 比較不同實作（JS vs Python）的效能差異

### 優缺點

| 優點 | 缺點 |
|------|------|
| 完整展示 ML 訓練流程 | JS 標量引擎極慢，僅能使用少數樣本 |
| 無需外部 ML 框架 | 網路架構簡單，辨識率有限 |
| 支援 MNIST 標準資料格式 | 無卷積層，忽略空間結構資訊 |
