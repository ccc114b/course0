# 自動微分測試範例

## 原理

本範例測試 nn0.js 的 Value 類別基本運算與自動微分功能。建立一個簡單的運算表達式 $c = a \times b + b^2$，然後計算 $c$ 對 $a$ 和 $b$ 的梯度，驗證反向傳播是否正確。

### 數學公式

**計算表達式**:
$$c = a \times b + b^2$$

**解析梯度**:
$$\frac{\partial c}{\partial a} = b$$
$$\frac{\partial c}{\partial b} = a + 2b$$

### 使用場景

- 驗證自動微分引擎的正確性
- 教學示範：展示計算圖與鏈式法則

### 優缺點

| 優點 | 缺點 |
|------|------|
| 簡單直觀，易於驗證 | 僅測試基本運算，未測試複雜網路 |
| 可手動驗證梯度數值 | 無邊界測試用例 |
