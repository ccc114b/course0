# test_gpt0.js — GPT 模型訓練與推理主程式

## 原理

本檔案是 gpt0.js 的入口程式，負責載入文本資料、建立 Tokenizer、初始化 GPT 模型、執行訓練與推論。示範完整的語言模型工作流程：資料 → 訓練 → 生成。

### 演算法步驟

1. 設定亂數種子，確保結果可重現
2. 讀取文本檔案，每行視為一個文檔
3. Fisher-Yates 演算法打亂文檔順序
4. 掃描所有字元建立詞彙表（字符級 Tokenizer）
5. 初始化 GPT 模型與 Adam 優化器
6. 執行 1000 步訓練
7. 使用訓練後的模型生成 20 筆新樣本

### 數學公式

**詞彙表建構**:
$$vocab = unique\_chars(docs)$$
$$BOS = |vocab|,\; vocab\_size = |vocab| + 1$$

### 使用場景

- 自訂文本資料集的語言模型訓練
- 展示完整的 ML pipeline（資料 → 訓練 → 推論）
- 實驗不同文本類型的生成效果

### 優缺點

| 優點 | 缺點 |
|------|------|
| 支援任意文本檔案輸入 | 僅支援字符級分詞 |
| 完整的訓練與推論流程 | 訓練速度受 JS 引擎限制 |
| 自訂隨機種子確保可重現 | 參數寫死，需修改原始碼調整 |
