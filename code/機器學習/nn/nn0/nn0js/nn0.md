# nn0.js — JavaScript 自動微分引擎與神經網路基礎元件

## 原理

nn0.js 是以 JavaScript 實作的自動微分引擎與神經網路元件庫。它透過 Value 類別建立計算圖（Computation Graph），支援反向傳播自動計算梯度。同時提供 Adam 優化器、線性層、Softmax、RMSNorm、交叉熵損失等元件，可用於建構與訓練小型神經網路。

### 核心架構

1. **Value 類別**: 實作運算節點與自動微分，支援加法、乘法、冪次、對數、指數、ReLU 等基本運算
2. **Adam 優化器**: 自適應學習率最佳化，含偏差修正與動量管理
3. **線性層 (linear)**: 矩陣乘法全連接層
4. **Softmax**: 數值穩定的多類別機率轉換
5. **RMSNorm**: 根均方正規化
6. **交叉熵損失 (cross_entropy)**: 使用 Log-Sum-Exp 技巧確保數值穩定性
7. **梯度下降 (gd)**: 封裝前向傳播、反向傳播與參數更新的完整訓練步驟

### 演算法步驟

**自動微分**:
1. 前向傳播時建立計算圖節點，記錄子節點與局部梯度
2. 反向傳播時 DFS 建立拓樸排序
3. 逆序遍歷，對每個子節點累加梯度（鏈式法則）

**Adam 優化**:
1. 更新一階動量 m = β1·m + (1-β1)·grad
2. 更新二階動量 v = β2·v + (1-β2)·grad²
3. 偏差修正 m̂ = m/(1-β1^t), v̂ = v/(1-β2^t)
4. 參數更新 p -= lr · m̂ / (√v̂ + ε)

### 數學公式

**鏈式法則**:
$$\frac{\partial L}{\partial x} = \frac{\partial L}{\partial y} \cdot \frac{\partial y}{\partial x}$$

**加權隨機取樣 (Weighted Choice)**:
$$P(i) = \frac{w_i}{\sum_j w_j}$$

### 使用場景

- 在 Node.js 環境中實作小型神經網路
- 理解自動微分與反向傳播的教學工具
- 快速原型開發 ML 模型

### 優缺點

| 優點 | 缺點 |
|------|------|
| JS 生態系，易於部署到網頁應用 | 純標量運算，大模型效率低 |
| 完整的自動微分實作 | 無 GPU 加速 |
| 模組化設計，易於擴展 | 僅支援 CPU 單執行緒運算 |
