# gpt0demo — Rust GPT 模型訓練與推理主程式

## 原理

本檔案是 Rust 實作 GPT 模型的主程式入口，負責完整的資料載入、預處理、模型初始化、訓練與推論流程。使用命令列引數指定輸入文本檔案，示範完整的語言模型 pipeline。

### 演算法步驟

1. 從命令列參數讀取資料檔案路徑
2. 讀取檔案，每行視為一個文檔，去除空白行
3. Fisher-Yates 隨機打亂文檔順序（固定種子 42）
4. 掃描所有字元建立字符級詞彙表（HashSet 去重）
5. 計算 BOS 標記 ID 與詞彙表大小
6. 初始化 GPT 模型與 Adam 優化器
7. 執行 1000 步訓練
8. 使用 temperature = 0.5 生成 20 筆樣本

### 數學公式

**詞彙表建構**:
$$vocab = sorted(unique\_chars(docs))$$
$$BOS = |vocab|,\; vocab\_size = |vocab| + 1$$

**學習率衰減**:
$$lr_t = lr_0 \times (1 - \frac{step}{num\_steps})$$

### 使用場景

- Rust 生態系的語言模型訓練與推理
- 高效能文字生成應用
- 教學用完整 ML pipeline 展示

### 優缺點

| 優點 | 缺點 |
|------|------|
| Rust 實作執行效率高 | 僅支援字符級分詞 |
| 完整 pipeline（資料→訓練→推論） | 模型大小固定，需修改原始碼調整 |
| 使用固定隨機種子確保可重現 | 需手動管理路徑與檔案 |
