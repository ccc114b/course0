# main.py — GPT 模型訓練與推論主程式（nn1tensor 版）

## 原理

本程式示範使用 nn1tensor 的 Tensor 引擎與 GPT 模型進行語言模型訓練與推論。透過 Tensor 的向量化運算，可以一次計算整個序列的 loss，大幅提升訓練效率。

### 演算法步驟

1. 載入資料集（人名列表）
2. 建立字符級 Tokenizer
3. 初始化 GPT 模型與 Adam 優化器
4. 訓練迴圈：
   - 批次準備輸入張量 (B=1, T=n) 與目標張量
   - 前向傳播計算 logits
   - Cross-Entropy 損失（全序列一次計算）
   - 反向傳播與 Adam 更新
   - 學習率線性衰減
5. 推論：自回歸逐 token 生成

### 數學公式

**向量化損失計算**:
$$Loss = \frac{1}{BT}\sum_{b=1}^B\sum_{t=1}^T -\log P(y_{b,t} | x_{b,1:t})$$

### 使用場景

- 向量化語言模型訓練示範
- 理解 Tensor 引擎相較 Value 引擎的效能優勢
- 教學用：展示批次處理的基本概念

### 優缺點

| 優點 | 缺點 |
|------|------|
| 向量化訓練，一次計算整段序列 | 批次大小固定為 1 |
| 使用 Tensor 引擎效能較佳 | 推論仍為逐 token 自回歸 |
| 完整的訓練與推論流程 | 無梯度裁剪、早停等進階技巧 |
