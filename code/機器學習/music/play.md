# MIDI 音符播放器

## 原理

本程式使用 `scamp` 音樂即時演奏函式庫，將文字格式的音符序列轉換為 MIDI 訊號並透過鋼琴音色播放。核心為**音符到 MIDI 編號的映射**：每個音名加上八度（如 `C4`）對應一個標準 MIDI 數值，其中 `C4 = 60`。

輸入檔案格式為成對的 `音符 時值`，以空白分隔。

### 演算法步驟

1. **讀取輸入檔**：讀取文字檔並以空白切分為 token 列表。
2. **成對解析**：依序讀取 `(音符, 時值)` 對，時值為浮點數表示秒數。
3. **音符轉換**：將音名與八度（如 `C#4`）轉為 MIDI 半音編號。
4. **即時播放**：使用 `scamp` 的 `play_note(midi_pitch, 音量, 時值)` 播放每個音符。
5. **錯誤處理**：若格式錯誤（非成對）、檔案不存在或音符解析失敗則回報。

### 數學公式

**MIDI 編號計算**:
$$\text{MIDI} = (\text{octave} + 1) \times 12 + \text{semitone\_index}$$

其中 `C=0, C#=1, ..., B=11`，八度編號 `C4` 的 `octave=4`，`(4+1)*12 + 0 = 60`。

## 使用場景

- 將自動生成的旋律文字檔轉為實際音訊
- 教學展示：串接馬可夫鏈生成器與播放器
- 快速試聽 MIDI 樂譜

## 優缺點

| 優點 | 缺點 |
|------|------|
| 即時播放，無需外部音訊檔案 | 依賴 scamp 函式庫（需 MIDI 驅動） |
| 支援升記號（#）音名 | 不支援降記號（b）標記 |
| 簡單的文字格式，易於編輯 | 單音播放，無和弦功能 |
