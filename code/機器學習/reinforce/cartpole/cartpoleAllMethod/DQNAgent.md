# DQN (Deep Q-Network) — CartPole-v1

## 原理

DQN 是 Deep Q-Learning 的代表性演算法，使用深度神經網路來逼近 Q 函數 $Q(s,a)$。核心創新包含兩項關鍵技術：

1. **目標網路（Target Network）**：使用一個獨立的目标網路 $Q_{\text{target}}$ 來計算 TD 目標，並定期從評估網路複製權重，以穩定訓練。
2. **經驗回放（Experience Replay）**：將轉移 $(s, a, r, s')$ 存入回放緩衝區，隨機抽樣進行訓練，打破序列資料間的相關性。

此實作使用 $\epsilon$-greedy 探索策略（$\epsilon = 0.001$），並在回放緩衝區累積足夠樣本後開始學習。

## 演算法步驟

1. 初始化評估網路 $Q(s,a;\theta)$ 與目標網路 $Q(s,a;\theta^-)$
2. 對每個時間步：
   - 以 $\epsilon$-greedy 策略選擇動作
   - 執行動作，將轉移存入回放緩衝區
   - 當緩衝區樣本達到閾值後：
     a. 從回放緩衝區隨機抽樣一個批次
     b. 計算 TD 目標：$y = r + \gamma \max_{a'} Q_{\text{target}}(s', a')$
     c. 計算損失：$\mathcal{L} = (y - Q(s,a;\theta))^2$
     d. 梯度下降更新評估網路
3. 回合開始時複製評估網路權重到目標網路

## 數學公式

**Q 值更新目標**:
$$y_i = r_i + \gamma \max_{a'} Q_{\text{target}}(s_i', a'; \theta^-)$$

**損失函數**:
$$\mathcal{L}(\theta) = \frac{1}{N}\sum_{i=1}^{N} \left(y_i - Q(s_i, a_i; \theta)\right)^2$$

## 使用場景

- 離散動作空間的強化學習任務
- 高維度狀態空間的問題（如圖像輸入）
- 作為深度強化學習的基礎演算法

## 優缺點

| 優點 | 缺點 |
|------|------|
| 離策略學習，樣本效率高 | 容易高估 Q 值（過度樂觀偏差） |
| 經驗回放打破資料相關性 | 對超參數敏感 |
| 目標網路穩定訓練 | 不適合連續動作空間 |
| 可處理高維度狀態空間 | 收斂性無理論保證 |
