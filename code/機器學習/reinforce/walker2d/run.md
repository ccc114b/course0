# PPO 推論腳本 — Walker2d-v5

## 原理

載入預訓練的 PPO 模型檢查點（包含模型權重與歸一化統計參數），在 Walker2d-v5 環境中進行推論視覺化。推論時使用歸一化後的觀測值輸入 Actor 網路，取得動作後執行。

此腳本還實作了攝影機跟隨功能，讓視角自動追蹤 Walker 的 x 軸位置，提供更好的觀察體驗。

## 演算法步驟

1. 載入檢查點（模型權重 + 歸一化統計 $(\mu_{\text{rms}}, \sigma_{\text{rms}})$）
2. 進入無限循環：
   - 歸一化觀測：$s_{\text{norm}} = (s - \mu_{\text{rms}}) / \sqrt{\sigma_{\text{rms}} + \epsilon}$
   - Actor 網路計算動作
   - 執行動作，更新攝影機位置
   - 若回合終止則重置環境

## 數學公式

**狀態歸一化**:
$$\tilde{s} = \frac{s - \mu_{\text{rms}}}{\sqrt{\sigma_{\text{rms}} + \epsilon}}$$

**動作**:
$$a = \text{Actor}(\tilde{s})$$

## 使用場景

- 展示訓練完成的 Walker2d 行走效果
- 視覺化模型在不同策略下的行為
- 錄製展示影片

## 優缺點

| 優點 | 缺點 |
|------|------|
| 攝影機自動跟隨 | 依賴特定檢查點格式 |
| 使用訓練時的歸一化統計 | 無命令列參數配置 |
| 簡潔的推論流程 | 無錄影功能 |
