# 擴散模型訓練（簡化版 — 失敗案例）

## 原理

訓練過程與標準 DDPM 相似：對隨機時間步 t，對圖片加噪後讓 UNet 預測所加噪聲。但與標準實作相比使用了更少的步數和簡化的網路結構。

### 與標準 DDPM 的差異

- 總步數 T=100（標準為 1000）
- 資料標準化：將 [0,1] 的輸入映射到 [-1,1]（標準作法）
- 簡化的 UNet 結構
- 無學習率排程

### 演算法步驟

1. 將影像從 [0,1] 縮放到 [-1,1]
2. 隨機取樣 t 和噪聲 ε
3. 計算加噪影像 x_t
4. 用 UNet 預測噪聲，計算 MSE 損失
5. 反向傳播更新參數

### 存在的問題

- 前向擴散排程僅 100 步，可能不足以收斂到純噪聲
- 生成時的部分實作錯誤（見 generate.md）

## 使用場景

- 教學：理解擴散模型實作中的要點
- 實驗：不同超參數效果的對比

## 優缺點

| 優點 | 缺點 |
|------|------|
| 訓練速度快 | 步數少導致品質下降 |
| 易於理解 | 缺少最佳實踐技巧 |
