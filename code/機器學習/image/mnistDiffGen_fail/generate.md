# 擴散模型生成（簡化版 — 失敗案例）

## 原理

本程式嘗試實作一個簡化版本的擴散模型生成，與標準 DDPM 相比有幾個關鍵差異（也是可能導致生成品質不佳的原因）：
- 擴散步數僅 100 步（標準為 1000 步）
- 逆向過程缺少正確的後驗變異數項
- 噪聲添加被乘以 0 實際關閉

### 演算法步驟

1. 從標準常態取樣 x_T
2. 對 t = T-1, ..., 1 迭代：
   a. 用 UNet 預測噪聲
   b. 根據簡化公式計算 x_{t-1}
   c. 噪聲項被乘以 0（無隨機性）
3. 將輸出從 [-1,1] 剪裁至 [0,1]

### 存在的問題

- 缺少正確的後驗變異數計算
- 逆向式中係數可能不正確
- 噪聲項被關閉可能導致品質下降

## 使用場景

- 理解擴散模型實作中的常見錯誤
- 對比正確與錯誤實作的效果差異

## 優缺點

| 優點 | 缺點 |
|------|------|
| 概念展示清楚 | 生成品質可能不佳 |
| 程式碼簡潔 | 缺少關鍵的變異數項 |
