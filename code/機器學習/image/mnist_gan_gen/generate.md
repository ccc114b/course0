# GAN 生成器（Generator）— 手寫數字生成

## 原理

生成對抗網路（GAN）由生成器（Generator）和判別器（Discriminator）組成。生成器的任務是從隨機噪聲中合成逼真的偽造影像，試圖欺騙判別器。訓練完成後，生成器本身就是一個完整的生成模型，可以將任意隨機向量 z 映射為逼真的影像。

本程式的生成器使用轉置卷積（Transposed Convolution）將 100 維的潛在向量逐步放大為 28×28 的灰階影像。

### 演算法步驟

1. 載入訓練好的生成器權重
2. 從標準常態分佈取樣潛在向量 z ~ N(0, I)，大小為 (N, 100)
3. 將 z 通過生成器網路：
   a. 全連接層映射到 7×7×64
   b. 重塑為 (64, 7, 7) 特徵圖
   c. 轉置卷積上採樣至 (32, 14, 14)，批歸一化 + ReLU
   d. 轉置卷積上採樣至 (1, 28, 28)，Tanh 激活輸出
4. 儲存生成的影像

### 數學公式

**生成器映射**:
$$G(z) = \tanh\left(\text{ConvTranspose}_2\left(\text{ReLU}\left(\text{BN}\left(\text{ConvTranspose}_1\left(\text{Reshape}(Wz + b)\right)\right)\right)\right)\right)$$

**潛在空間取樣**:
$$z \sim \mathcal{N}(0, I), \quad z \in \mathbb{R}^{100}$$

## 使用場景

- 影像生成（本程式的 MNIST 數字生成）
- 資料擴增
- 創意內容生成
- 無監督學習中的特徵提取

## 優缺點

| 優點 | 缺點 |
|------|------|
| 生成速度快（一次前向傳播） | 訓練不穩定，容易模式崩潰 |
| 生成影像銳利 | 無法像擴散模型計算似然 |
| 潛在空間可進行插值 | 需要對抗訓練，收斂困難 |
