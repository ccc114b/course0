# MNIST 額外測試圖片下載工具

## 原理

這是一個簡單的資料提取工具，從 MNIST 測試集中選取特定數字的影像並儲存為 PNG 檔案，用於後續手動測試識別模型。

### 演算法步驟

1. 載入 MNIST 測試集（transform 僅轉為 Tensor）
2. 遍歷測試集，尋找數字 3、6、8
3. 每種數字提取前 2 張，儲存為 PNG（值域從 [0,1] 映射回 [0,255]）

## 使用場景

- 準備測試資料用於驗證模型
- 建立小型自訂測試集

## 優缺點

| 優點 | 缺點 |
|------|------|
| 簡單快速 | 固定的輸出格式 |
| 自動從 MNIST 提取 | 需手動指定需要的數字 |
