# CNN MNIST 手寫數字推論

## 原理

使用訓練好的卷積神經網路（CNN）對單張手寫數字圖片進行分類。模型由兩層卷積加兩層全連接層組成，輸出 10 個類別的 logits，取最大值對應的類別作為預測結果。

### 網路結構

- Conv1: 1→32 通道，3×3 卷積 + ReLU + 2×2 MaxPool
- Conv2: 32→64 通道，3×3 卷積 + ReLU + 2×2 MaxPool
- FC1: 64×5×5 → 128，ReLU + Dropout(0.5)
- FC2: 128 → 10（輸出 logits）

### 演算法步驟

1. 載入模型權重，設為評估模式
2. 讀取輸入圖片，預處理：灰階化 → 縮放至 28×28 → 正規化至 [-1, 1]
3. 前向傳播，得到 10 維輸出向量
4. 取 argmax 得到預測數字

### 數學公式

**前向傳播與分類**:
$$\hat{y} = \arg\max_i \, (W_2 \cdot \text{ReLU}(W_1 \cdot \text{Pool}(\text{ReLU}(W_c * x))) + b)$$

## 使用場景

- 單張圖片的即時數字識別
- 部署於邊緣裝置的輕量級分類器

## 優缺點

| 優點 | 缺點 |
|------|------|
| 推論速度快 | 對抗性攻擊脆弱 |
| 模型輕量（約 1.2M 參數） | 僅處理 28×28 灰階輸入 |
| 準確率高（>98%） | 無資料擴增時的泛化有限 |
