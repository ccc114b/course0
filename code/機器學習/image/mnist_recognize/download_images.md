# MNIST 測試圖片下載工具

## 原理

從 MNIST 測試集中取出前 10 張影像（每個數字各一張），儲存為 PNG 檔案，方便進行模型推論測試。

### 演算法步驟

1. 載入 MNIST 測試集
2. 迭代前 10 筆資料
3. 將 Tensor 轉回 PIL 影像（值域從 [0,1] 映射到 [0,255]）
4. 以 `{標籤}_{索引}.png` 格式儲存

## 使用場景

- 快速準備測試資料
- 驗證訓練好的分類模型

## 優缺點

| 優點 | 缺點 |
|------|------|
| 無需外部資料 | 測試樣本量少 |
| 自動標註檔名 | 僅供基本驗證用 |
