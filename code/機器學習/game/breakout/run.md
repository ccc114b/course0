# Breakout DQN 遊玩器

## 原理

本程式載入以 **Deep Q-Network (DQN)** 訓練好的模型，在 Atari Breakout 環境中進行推論與遊玩。DQN 是深度強化學習的經典方法，使用卷積神經網路將遊戲畫面（4 幀堆疊灰階圖）映射到每個動作的 Q 值，再以貪婪策略（Greedy Policy）選擇最高 Q 值的動作執行。

與訓練階段不同，遊玩時不進行 ε-greedy 探索，完全利用已學到的策略。

### 演算法步驟

1. **載入模型**：讀取 `.pt` 權重檔，支援純 `state_dict` 或含中繼資料的 checkpoint 格式。
2. **建立環境**：使用 `gymnasium` 建立 `ALE/Breakout-v5`，套用 Atari 預處理（灰階、縮放至 84×84、幀跳躍、4 幀堆疊）。
3. **貪婪推論**：對觀測值取 argmax Q 值作為動作。
4. **回合執行**：反覆執行動作、累積獎勵，直到回合終止或截斷。
5. **統計輸出**：計算多回合的平均、最高、最低獎勵。

### 數學公式

**Q 值近似**:
$$Q(s, a; \theta) \approx \mathbb{E}\left[ \sum_{t=0}^\infty \gamma^t r_t \mid s_0 = s, a_0 = a \right]$$

**貪婪動作選擇**:
$$a^* = \arg\max_a Q(s, a; \theta)$$

## 使用場景

- 驗證訓練完成的 DQN 模型效果
- 錄製 AI 遊玩影片作為展示
- 人機對戰或純計算分數的基準測試

## 優缺點

| 優點 | 缺點 |
|------|------|
| 支援自動尋找最新 checkpoint | 僅支援貪婪推論，無探索 |
| 可錄影與控制播放速度 | 無 Monte Carlo 樹搜尋等進階規劃 |
| 相容 train.py 的所有儲存格式 | 無隨機種子控制（結果可能浮動） |
