CPU 架構

CPU 基本結構

CPU（中央處理器）由多個功能模組組成：控制單元（CU）負責指令擷取、解碼和產生控制訊號；運算單元（ALU）執行算術邏輯運算；暫存器檔案（Register File）提供最快存取；快取記憶體（L1/L2/L3）減少記憶體延遲。

控制單元

控制單元是 CPU 的大腦，協調各部件運作：

  • 指令擷取（Fetch）— 從記憶體讀取指令，更新 PC
  • 指令解碼（Decode）— 解釋操作碼，讀取暫存器操作數
  • 控制訊號產生 — 決定 ALU 運算、資料流向、暫存器寫入
  • 分支處理 — 計算分支目標，管理分支預測

指令執行週期

經典的五階段指令週期：

  1. IF (Instruction Fetch) — 從記憶體取指令
  2. ID (Instruction Decode) — 解碼並讀取暫存器
  3. EX (Execute) — ALU 執行運算或計算位址
  4. MEM (Memory Access) — 載入/儲存資料
  5. WB (Write Back) — 結果寫回暫存器

現代 CPU 特徵

  • 管線化 — 多條指令同時處於不同階段
  • 超純量 — 每個時脈週期發射多條指令
  • 亂序執行 — 動態排程最大化執行單元利用率
  • 分支預測 — 猜測分支方向避免管線停頓
  • 多核心 — 多個 CPU 核心共享快取
