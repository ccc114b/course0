HTML 的發展歷史

HTML (HyperText Markup Language) 由 Tim Berners-Lee 於 1990 年發明，作為 CERN 的資訊共享系統。

  • HTML 1.0 (1991) — 約 18 個標籤，Web 的起點
  • HTML 2.0 (1995) — 第一個標準化版本，加入表單與圖片
  • HTML 4.01 (1999) — 引入 CSS 支援
  • HTML5 (2014) — 語義化標籤、多媒體、Canvas、WebSocket

HTML 文件基本結構

每個 HTML 文件以 <!DOCTYPE html> 宣告開始，包含 <head> 與 <body> 兩大部分。

<!DOCTYPE html>
<html lang="zh-TW">
<head>
<meta charset="UTF-8">
<title>網頁標題</title>
</head>
<body>
<h1>Hello World</h1>
</body>
</html>

常見 HTML 標籤

HTML 提供豐富的標籤來組織內容：

  • 文字: <h1>-<h6>, <p>, <strong>, <em>
  • 清單: <ul>, <ol>, <li>
  • 表格: <table>, <tr>, <td>, <th>
  • 表單: <form>, <input>, <select>, <textarea>
  • 語義化: <header>, <nav>, <main>, <article>, <section>, <footer>

HTML5 新特性

HTML5 引入了許多革命性功能：多媒體標籤 (<video>, <audio>)、圖形繪製 (Canvas, SVG)、本地儲存 (LocalStorage, SessionStorage)、Web Workers 和 WebSocket。
