資料庫

資料儲存與查詢系統

資料庫發展歷程

  • 層次與網路資料庫 (1960s-1970s) — 最早的資料庫系統
  • 關聯式資料庫 (1970s-現在) — Edgar F. Codd 提出關聯模型，Oracle、MySQL、PostgreSQL 普及
  • NoSQL 資料庫 (2000s-現在) — 因應 Web 2.0 和大資料時代而興起
  • 多模資料庫 (2010s-現在) — 支援多種資料模型

關聯式資料庫 (RDBMS)

資料以表格 (Table) 組織，透過鍵 (Key) 建立關係，使用 SQL 進行操作。

CREATE TABLE users (
id INT PRIMARY KEY AUTO_INCREMENT,
name VARCHAR(100) NOT NULL,
email VARCHAR(255) UNIQUE NOT NULL,
age INT,
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

SQL 基本操作

-- 查詢
SELECT * FROM users WHERE age >= 18 ORDER BY created_at DESC;

-- 新增
INSERT INTO users (name, email) VALUES ('王小明', 'wang@example.com');

-- 更新
UPDATE users SET age = 26 WHERE name = '王小明';

-- 刪除
DELETE FROM users WHERE age < 18;

NoSQL 資料庫

放棄嚴格的表格結構，提供更高擴展性。常見類型包括：

  • 文件資料庫 (MongoDB) — JSON-like 文件儲存
  • 鍵值儲存 (Redis) — 高速記憶體資料庫
  • 列式資料庫 (Cassandra) — 大規模分散式儲存
  • 圖形資料庫 (Neo4j) — 關聯網路分析
