xv6

教育用作業系統

什麼是 xv6？

xv6 是 MIT 為作業系統課程開發的教學用 Unix-like 作業系統，是 Unix V6（PDP-11）的現代化移植版本。xv6 使用 ANSI C 實作，支援 x86 和 RISC-V 架構，程式碼精簡（約 9000 行），非常適合學習作業系統核心概念。

核心功能

  • 行程管理：fork、exec、wait、exit 系統呼叫
  • 記憶體管理：分頁式虛擬記憶體
  • 檔案系統：日誌式檔案系統（log）
  • 管道與訊號：行程間通訊機制
  • 驅動程式：控制台、磁碟、網路

程式碼架構

檔案功能
proc.c行程管理與排程
vm.c虛擬記憶體管理
fs.c檔案系統實作
syscall.c系統呼叫處理
trap.c中斷與異常處理
