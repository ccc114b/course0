虛擬機 (Virtual Machine)

軟體模擬的計算機

什麼是虛擬機？

虛擬機（Virtual Machine）是一種軟體模擬技術，通過軟體模擬硬體或特定指令集，在單一硬體平臺上執行多個獨立的作業系統與應用環境。虛擬化技術是雲端運算的基礎。

虛擬化的類型

系統虛擬化

模擬完整硬體平臺，讓客端作業系統在隔離環境中執行。代表：QEMU、VMware、VirtualBox、Hyper-V。

程序虛擬化

模擬特定指令集，讓同一平臺上執行不同架構的程式。代表：JVM、CLR、WebAssembly Runtime。

作業系統虛擬化

共享主機核心，隔離程序與資源。代表：Docker 容器。

Hypervisor

  • Type 1（裸機型）：直接運行在硬體上，如 VMware ESXi、KVM
  • Type 2（託管型）：運行在主機 OS 之上，如 VirtualBox
