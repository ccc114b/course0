RISC-V

開放指令集架構的未來

什麼是 RISC-V？

RISC-V 是從 UC Berkeley 發起的開放標準指令集架構（ISA），與 ARM、x86 不同，完全免費開放。這種開放性使其在學術、嵌入式、雲端運算領域迅速獲得關注。

特點

  • 開放標準：無需授權費用
  • 模組化設計：基本指令集 + 擴充套件
  • 精簡：基礎指令集易於學習和實現
  • 擴展性：支援自訂指令
  • 社群活躍：開源生態系統

基本整數指令集

類別指令說明
算術add, sub, mul加、減、乘
邏輯and, or, xor位元運算
移位sll, srl, sra邏輯/算術移位
分支beq, bne, blt條件跳躍
載入/儲存lw, sw, lb, sb記憶體存取
跳躍jal, jalr函數呼叫

擴展套件

M：整數乘除法；A：原子操作；F/D：單/雙精度浮點；C：壓縮指令；V：向量運算。
