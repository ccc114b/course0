TCP/IP 協定

網際網路的基礎通訊協定

什麼是 TCP/IP？

TCP/IP 是網際網路的核心通訊協定集，定義資料如何在網路中傳輸、路由和接收。分層架構讓網路通訊模組化且可擴展。

四層模型

應用層

HTTP、FTP、SMTP、DNS 等協定。

傳輸層

TCP：面向連線、可靠傳輸；UDP：無連線、適合即時應用。

網路層

IP 協定負責資料包路由與轉發。

網路介面層

乙太網路、Wi-Fi 等實體網路介接。
