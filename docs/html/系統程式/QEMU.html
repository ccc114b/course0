QEMU

硬體虛擬化與模擬器

什麼是 QEMU？

QEMU 是由 Fabrice Bellard 開發的開源機器模擬器與虛擬化軟體。它可以模擬多種 CPU 架構（x86、ARM、RISC-V 等），並支援透過 KVM 進行硬體加速虛擬化。QEMU 是嵌入式開發、作業系統教學和雲端基礎設施的重要工具。

主要功能

  • 系統模擬：模擬完整計算機系統，包含 CPU、記憶體、周邊設備
  • 使用者模式模擬：在一個架構上執行另一個架構的程式
  • KVM 加速：透過 Linux KVM 達到接近原生的效能
  • 快照與遷移：支援虛擬機快照與即時遷移
