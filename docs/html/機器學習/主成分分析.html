主成分分析 (PCA)

最常用的維度降低技術，透過線性變換將高維資料映射到低維空間

原理

主成分分析（Principal Component Analysis）是最常用的維度降低技術。其步驟為：標準化：將資料標準化（均值為 0，標準差為 1）。共變異矩陣：計算特徵間的共變異矩陣。特徵分解：找出特徵值和特徵向量。選擇主成分：選擇解釋最多變異的向量（特徵值最大的方向）。投影：將資料投影到主成分上，得到降維後的表示。PCA 保證投影後資料的變異量最大。

from sklearn.decomposition import PCA pca = PCA(n_components=2) X_pca = pca.fit_transform(X) print(f"解釋變異量: {pca.explained_variance_ratio_}") print(f"累積解釋變異量: {sum(pca.explained_variance_ratio_)}")

優點與應用

降低維度：減少計算成本和儲存空間，緩解維度詛咒。 去除雜訊：保留主要變異方向，捨棄可能由雜訊引起的小變異方向。 視覺化：將高維資料降為 2D 或 3D 便於可視化分析。 去相關：去除特徵間的相關性，得到正交的主成分。PCA 廣泛應用於基因數據分析、圖像壓縮、人臉識別（特徵臉）以及機器學習前的預處理步驟。

選擇主成分數量

通常根據累積解釋變異量選擇主成分數量，一般保留 95% 以上的變異量。可使用「肘部法」（Scree Plot）觀察特徵值的下降曲線，選擇曲線轉折點處的主成分數。PCA 假設資料的主要變異方向包含最重要的資訊。若資料的結構不主要體現在變異量上（如聚類結構），PCA 可能不是最佳選擇。
